# metriqchem
We have created a toy model of an electronic system without nuclear-electron or electron-electron interactions, based on atomic 
orbitals that specifies a spatial metric. Please refer to Equation 2.10 in the link below.
This program is a toy code that implements a Hamiltonian based on a spatial metric.

## Requirements
- Python3.10.10
- PySCF==2.12.1
- jax==0.6.2
- numpy==1.26.4
